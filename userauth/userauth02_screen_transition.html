Shared Screens  共通画面

These screens are used by multiple flows. Their appearance is controlled by the message_id parameter passed from the backend.

ログイン
/login
Entry point for all authenticated flows. Links to Register and Reset.
message.html
/message
Shows email-sent confirmation or error messages for all flows (REGISTER / RESET / CALL / INVITE / UPDATE-EMAIL).
error.html
/error
System error page. Displays error code and description. Used when any backend exception is raised.
